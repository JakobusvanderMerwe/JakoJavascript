Javascript Display Possibilities

innerHTML - writes into an HTML elements.
To access an HTML element use document.getElementById(id)

<p id="example"></p>
<script> document.getElementById("example").innerHTML = 5 + 6; </script>

This returns 11 within html

document.write()

Writes directly into html without needing to link to an HTML element

<script>document.write(5+7)</script> This will just show the result on the page.

Variables

Variables ar containers for storing data: let - for if the variable an change vs const for when the variable will remain the same.

const price1 = 5;
const price2 = 6;
let total = price1 + price2;

// You can create a constant array:
const cars = ["Saab", "Volvo", "BMW"];

// You can change an element:
cars[0] = "Toyota";

// You can add an element:
cars.push("Audi");


You can also change properties within the const and add properties (car.owner)
